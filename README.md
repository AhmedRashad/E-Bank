## This application enables Users to:

## The Admin can:

### View all application data on the dashboard

### Control all statuses users:

        * Active the users
        * Reject the users
        * Pend the users

### Control all accounts status of users:

        * Active the accounts
        * Reject the accounts
        * Pend the accounts
